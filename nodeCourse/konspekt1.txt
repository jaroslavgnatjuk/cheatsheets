npm repo <repo name> - откроет в браузере ссылку на гитхаб модуля
npm home <repo name> - откроет в браузере ссылку на домашний сайт модуля
npm list - список пакетов

NODE_PATH - переменная окружения для поиска модулей

octolinker, octotree - плагины хрома для удобной работы с гитхабом

const {Server} = require('http'); - деструктуризация обьекта http, берем свойство Server = require('http').Server;

порядок выполнения:
1. js выполняются последовательно js-инструкции
2. операции ввода-вывода, колбэки
process.nextTick и promise выполняются после js инструкций, но до операций ввода-вывода
setImmediate - срабатывает асинхронно как получится, может быть до ввода-вывода, а может после

модуль trace - для вывода полного стектрейса при асинхронных операциях
удобно для разбора ошибок
модуль clarify - убирает из стектрейса указание ссылок на внутренние файлы ноды

эти модули подключать следует 1 раз в самом начале, как вариант проверять переменную окружения:
if (process.env.DEV_TRACE) {
  Error.stackTraceLimit(1000);
  require('trace');
  require('clarify');
}
